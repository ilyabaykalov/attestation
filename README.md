# React template

> Шаблон react приложения

## Используемый стек технологий

- React.js
- TypeScript
- Axios

## Установка и запуск

### Установка переменных окружения в файл .env
> (Пример файла: [.env](./.env.examle))

### Установка зависимостей package.json

> npm install

### Запуск сервера

> npm run start
